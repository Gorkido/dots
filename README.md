# Gorkido/Dots
My linux configurations with a installation script.
  
## installation
```
sudo pacman -S --noconfirm git && cd ~ && rm -rf dots/ && git clone https://github.com/Gorkido/dots.git && cd dots/ && chmod +x install.sh && ./install.sh --install
```
